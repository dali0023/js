Variable:

Rules Ex with Data types:
  1. JavaScript is CASE SENSITIVE
  2. Var is not required, but using is it best practice
  3. Names must start with a letter, using _, $ in first not good practice.
  4. variable names can contain:
    1. letters- x,y,i
    2. numbers- x20
    3. _ (underscore)- _car, _x, my_car
    4. $-
  5. variables cannot start with a number
    1. 99flavours (wrong)
    2. flavours99 (Correct)
  6. Variables start with lowercase letter: myCar
  1. var a;
  2. var year, month, day;
  3. var age = 18; (number)
  4. var a = 1, b = 2, c = a + b; // one line
  5. string
    var name = "Jane";
    var d = 1 + 2 + "3"; //"33" [number+str = str]
    var x = "Volvo" + 16 + 4; //Volvo164.
    Js evaluates left to right.
  6. var name = {first:"Jane", last:"Doe"}; (Object)
  7. var truth = false; (boolean)
  8. var sheets = ["HTML","CSS","JS"]; (Array)
  9. var a; typeof a; (undefined)
  10. var a = null; (null)
  11. let z = 'zzz'; //local variable
